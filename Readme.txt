
<Data information>
pointgroup8.csv: Lattice-like grid points.
dist_pointgroup8.csv: Distance matrix among points on the grid.
kawamutsu_dist_samplingpoints2.csv: Distance matrix generated from dat_in_japan.csv

<How to run Simulation>
RunSimulation.R simulates the distribution formation process.
After the simulation, Posterior.R calculates the posterior distribution of each parameter.
